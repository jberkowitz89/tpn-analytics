# TPN-Analytics
An analytics web app for the [TPN Yahoo Fantasy](https://football.fantasysports.yahoo.com/league/tauphisammy/2023) league. 

This app will show the best, worst and most mediocre fantasy players for the history of the TPN fantasy league, going back to its inception in 2009 through present day.

Built in Python and using the [YFPY](https://yfpy.uberfastman.com/) Package.
